# thunhap.online

## Development

1. Install [Node.js](https://nodejs.org/en/download/)
2. Install [Yarn 1.x](https://classic.yarnpkg.com/en/docs/install)
3. Install project dependencies

```bash
yarn
```

4. Link to Supabase

```bash
yarn supabase link --project-ref ********************
```

5. Run the development server

```bash
yarn dev
```
